CSS Grid

Grid

A grid is a set of intersectiong horizontal and vertical lines defining columns and row, and elements can be placed onto this grid within the column and row lines. Together with flxbox, grid is a powerful way to create an intricate CSS layout.

Features:

  • Fixed and flexible track sizes
    • Fixed, e.g.: px
    • Flexible, e.g.: % and fr
  • Item placement
    • by exact location using line numbers, names, or by targeting an area of the grid.
    • by algorithm
  • Creation of additional tracks to hold content
    • Grid layout specification is flexible enough to add additional rows and columns when needed. There's even features to add as many as will fit.
  • Alignment control
  • Control of overlaping content
    • More than 1 item can be placed in 1 grid cell/area, as well as they can be partially overlap. Layering is controled by z-index property.
