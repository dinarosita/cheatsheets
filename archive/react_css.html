CSS Modules of Create-React-App

CSS Modules:

  • Create-React-App's built-in feature that allows us to scope styles into components. Behind the scene code transformation that allow us to attach css file to specific components.
  • File extension: module.css

Becoming like this:

  • components 🗁 > layout 🗁
    • MainNavigation.module.css
    • MainNavigation.js
    •

Steps

  • To the folder of ComponentName.js file, add: ComponentName.module.css
  • Import css file to the js file: import classes from "./MainNavigation.module.css";
  • Add classname started with classes. for example className = {classes.header} to the element/component.
  • Style it in the module.css file like this: .header { color: red; }. This header styling will only affect the class header within the MainNavigation component.
  • If the module.css is given by a third party, just adapt the js file by adding respective class names.
