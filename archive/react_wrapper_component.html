React: Wrapper Component

Wrapper Component

Wrapper component is a component that we can use to wrap around another component to give them certain feature. It's very handy to apply a style that is used frequently accross the website. For example, styling content box and border, and cards.

Children

To pass all the components wrapped within the wrapper, use children prop. It's a build in prop by react which mean to pass all the content that is wrapper with the wrapper.

Card.js

import classes from "./Card.module.css"

function Card(props) {
  return (
    <div className={classes.card}>
      {props.children}
    </div>
  )
}

MainApp.js

import Card from "../ui/Card"

function MainApp() {
  return (    
    <Card>
      <h1>Title</h1>
      <p>First paragraph</p>
      <p>Second paragraph</p>
    </Card>
  );
}
