JavaScript: Function.prototype.call()

Function.prototype.call()

call()
This method allows a function or method belonging to one object to be assigned and called for a different object.
It provides a new value this to the function/method.
With call(), we can write a method once, then inherit it in another object without having to rewrite the method for the new object.
It's similar to apply() method, with fundamental difference:
  • apply() accepts a single array of arguments.
  • call() accepts an argument list.
Syntax
call()
call(thisArg)
call(thisArg, arg1)
call(thisArg, arg1, arg2)
call(thisArg, arg1, ..., argN)
Parameters
  • thisArg (optional): The value to use as this when calling function. Check reference for further explanation.
  • arg1, arg2, ..., argN (optional): Arguments for the function
Return value:
  • The result of calling the function with the specified this value and arguments.
Example
function Product(name, price) {
    this.name = name;
    this.price = price;
}
    
function Food(name, price) {
    Product.call(this, name, price);
    this.category = 'food';
}

console.log(new Food('cheese', 5).name);
Breakdown:
  • Let's start with function Product(name, price) {...}
  • function Product is a function that's taking 2 arguments: name and price.
  • this inside the function shows that the function is an object method. A mthod own by the object. And this refers to that object.
  • this.name = name; instructs the name argument to be put in the name property of the object. Same deal with price.
  • The function creates:
    { name: argName, price: argPrice }
  • Now the missing piece is, what's this very object we are talking about? Function is not positioned inside an object already. We need to find away for an object to be called in place to house this function.
  • This is were call() is needed. Remember the syntax:
    call(thisArg, arg1, arg2)
  • Let's look up again the discussion on this page on call()
    • object1.method1()                 // Calling function 1    
object1.method1.call(object2)     // Calling function 1 but on object 2
  • Then:
    • Product(name, price) becomes something like:
    • Product.call(object2, name, price), and object2 will be referred with this:
    • Product.call(this, name, price)
