Logical Operators

There are 4 logical operators in JavaScript:

  • || (OR)
  • && (AND)
  • ! (NOT)
  • ?? (Nullish Coalescing)

They can be applied to values of any type with results of any type.

Truthy and Falsy

Every value in JavaScript has an inherent boolean value: truthy or falsy.

Falsy values

  • null
  • undefined
  • Boolean: false
  • Number: 0, -0, NaN
  • BigInt: 0n
  • String: '', "", ``

Everything else is truthy

  • Boolean: true
  • Any numbers, except for zeros and NaN
  • Any bigint, except for zero
  • Any strings, except for the empty ones
  • Any symbols
  • Any objects, including empty array [], empty object{}, and empty function () {}

|| (OR)

operand1 || operand2
operand1 || operand2 || operand3 and so on

OR: If any operand is truthy, then it's truthy.

Return value: first truthy or last falsy when there's no truthy.

Operator || does the following:

  • It evaluates operands from left to right.
  • For each operand, the operand is converted into its boolean. If the boolean is true, it stops and returns the original value of the operand.
  • If it reaches the end and all were false, it returns the original value of the last operand.

The classic boolean operands

boolean1 || boolean2
true  || true   // true
true  || false  // true
false || true   // true
false || false  // false

With any data type operands

true || true    // true
true || 5       // true
5 || true       // 5 (truthy)
"A" || 5        // "A" (truthy)

true  || false  // true
false || 5      // 5 (truthy)
5 || false      // 5 (truthy)
"A" || 0        // "A" (truthy)

false || false  // false
false || 0      // 0 (falsy)
0 || false      // false
"" || 0         // 0 (falsy)
0 || "" || null // null (falsy)

&& (AND)

operand1 && operand2

&& (AND): Both operands must be true to return true

true  || true   // true
true  || false  // false
false || true   // false
false || false  // false
