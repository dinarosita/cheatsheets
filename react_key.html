React: Key

List and Key

Each child in a alist should have a unique "key" prop. This is a requirement by react that is needed to update and render lists efficiently.

<ul className={classes.list}>
    {props.meetups.map((meetup) => {
        return (
        <MeetupItem
            key={meetup.id}
            meetup={meetup}
        />
        );
    })}
</ul>
