DevTools Introduction

DevTools: developer tools. Used to inspect elements.

How to access DevTools:

  • Press F12, or:
  • Right click on element, choose inspect element

Open chrome browser with DevTools open automatically for each window:

  • From terminal, open chrome with additional flags:
    $ google-chrome --auto-open-devtools-for-tabs
  • Only works if there's no chrome already opened
  • Future opening html code with xdg-open filename.html will open in new chrome window without DevTools pre-opened
  • Note, chrome opened from terminal keeps erroering. with or without the tags
