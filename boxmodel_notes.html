CSS Box Model Notes

Box model basic:

  • Padding: space around the text
  • Border: the box's outline
  • Margin: space outside the border, overlap with surrounding margins

Useful notes:

  • height and weight
    to define the text box size.
  • box-sizing: border-box;
    To make height and width applied to box border (outer) size.
  • display: inline-block;
    To make the box wrap the text instead of the whole length of line
.classname {
padding: 10px;
border: 2px solid pink;
margin: 20px;

height: 100px;
width: 200px;
box-sizing: border-box;
display: inline-block;
}

Some HTML elements, such as <a>, <span>, <am>, and <strong> use inline as their outer display type by default.
