JavaScript: Object Constructor

Object constructor

Constructor function
Let's using example to learn the syntax
function Person(first, last, year, place) {
    this.firstName = first;
    this.lastName = last;
    this.birthYear = year;
    this.location = place;
};
Constructor function
  • In example above: Person
  • Constructor functions is commonly named with an uppercase first later.
Arguments: object property values
  • In example above: first, last, year, and place.
  • To be placed in corresponded object properties.
Object property
  • In example above: this.firstName, this.lasName, this.birthYear, and this.location
this
  • In a construction function, this doesn't have a value. It's substitute for a new object.
  • The value of this will become the new object when a new object is created using the constructor function.

Object types (blueprints) (classes)

Object types. Blueprint, classes
  • Think of blueprint. How to create many objects of the same type.
  • Object type can be created using object constructor function, like function Person() above.
  • Objects of the same type are created by calling the constructor function with the new keyword.
function Person(first, last, year, place) {
    this.firstName = first;
    this.lastName = last;
    this.birthYear = year;
    this.location = place;
};

const father = new Person("Bruce", "Lee", 1950, "Toronto");

father;  // Person {firstName: 'Bruce', lastName: 'Lee', birthYear: 1950, location: 'Toronto'}

Blueprint

Creating many objects of the same type
With object constructor function, coupled with new keyword to call the function, we can create many objects of the same type. Object constructor function is the blueprint or "object type".
Examples using object constructor function above
const father  = new Person("Bruce", "Lee", 1950, "Toronto");
const mother  = new Person("Maria", "Santa", 1951, "Toronto");
const sister  = new Person("Puff", "Tail", 1985, "Niagara");
const brother = new Person("Dodo", "Bird", 1983, "Chatham");

father;   // Person {firstName: 'Bruce', lastName: 'Lee', birthYear: 1950, location: 'Toronto'}
mother;   // Person {firstName: 'Maria', lastName: 'Santa', birthYear: 1951, location: 'Toronto'}
sister;   // Person {firstName: 'Puff', lastName: 'Tail', birthYear: 1985, location: 'Niagara'}
brother;  // Person {firstName: 'Dodo', lastName: 'Bird', birthYear: 1983, location: 'Chatham'}
