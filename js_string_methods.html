JavaScript String Methods and Properties

String Length

str.length

The length property returns the length of a string.

let str = "Kitten sings a song.";
let length = str.length;                // 20

Extracting String Parts

Javascript counts position from 0. Then, last position will be str.length-1.

There are 3 methods to extract parts of a string:

  • slice(start, end)
  • substring(start, end)
  • substr(start, length)
