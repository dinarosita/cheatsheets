JavaScript: Sorting Arrays

Alphabetic sort()

sort()
Sorts an array alphabetically
const catRhyme = ["rat", "bat", "mat", "cat", "hat"];
catRhyme.sort();    // (5) ['bat', 'cat', 'hat', 'mat', 'rat']
catRhyme;           // (5) ['bat', 'cat', 'hat', 'mat', 'rat']
reverse()
Reverse the order of an array
const catRhyme = ["rat", "bat", "mat", "cat", "hat"];
catRhyme.reverse(); // (5) ['hat', 'cat', 'mat', 'bat', 'rat']
catRhyme;           // (5) ['hat', 'cat', 'mat', 'bat', 'rat']
Sorting in reverse order? sort() then reverse()
const catRhyme = ["rat", "bat", "mat", "cat", "hat"];
catRhyme.sort();    // (5) ['bat', 'cat', 'hat', 'mat', 'rat']
catRhyme.reverse(); // (5) ['rat', 'mat', 'hat', 'cat', 'bat']
catRhyme;           // (5) ['rat', 'mat', 'hat', 'cat', 'bat']

Numeric sort()

sort() doesn't sort numerically
Method sort() sorts alphabetically. It compares digit by digit like in the string. For example, when comparing 10 with 2, first it will compare "1" with "2". Therefore 10 is regarded to be smaller than 2.
const numbers = [30, 5, 200, 3, 50];
numbers.sort();      // (5) [200, 3, 30, 5, 50]
"Compare function"
function(a, b){return a - b}
Compare function returns 3 kind of value:
  • negative
  • zero
  • positive
Sorting numeric array
sort(function(a, b){return a - b})
Sorting according to the returned value:
  • negative: a is sorted before b
  • zero: no changes in order
  • positive: b is sorted before a
const numbers = [30, 5, 200, 3, 50];
numbers.sort(function(a, b){return a - b});      // (5) [3, 5, 30, 50, 200]
Reverse sorting numeric array
sort(function(a, b){return b - a})
Simply switch the return into b - a
const numbers = [30, 5, 200, 3, 50];
numbers.sort(function(a, b){return b - a});      // (5) [200, 50, 30, 5, 3]
Random sorting numeric array
sort(function(a, b){return 0.5 - Math.random()})
We can also use this to randomize array too by randomizing negative and positive value: b - a
const numbers = [3, 5, 30, 50, 200];
numbers.sort(function(a, b){return 0.5 - Math.random()});      // (5) [30, 5, 3, 200, 50]
numbers.sort(function(a, b){return 0.5 - Math.random()});      // (5) [5, 200, 50, 30, 3]
numbers.sort(function(a, b){return 0.5 - Math.random()});      // (5) [200, 5, 30, 50, 3]
However the result is not purely random because there's some favoring happening. Correct (and most popular) for random sorting is called Fisher Yates Method.
Fisher Yates Method for random sorting
const numArray = [3, 5, 30, 50, 200];

for (let i = numArray.length -1; i > 0; i--) {
    let j = Math.floor(Math.random() * i)
    let k = numArray[i]
    numArray[i] = numArray[j]
    numArray[j] = k
}

numArray;     // (5) [50, 30, 200, 5, 3]

sort() for min and max

Using index [0] and [arr.length-1]
There's no built in method, but once the array is sorted, we can obtain the lowest and highest value using index.
const min = sortArr[0];
const max = sortArr[sortArr.length - 1];
For example
const numbers = [30, 5, 200, 3, 50];
numbers.sort(function(a, b){return a - b});      // (5) [3, 5, 30, 50, 200]
const min = numbers[0];
const max = numbers[numbers.length - 1];

numbers;  // (5) [3, 5, 30, 50, 200]
min;      // 3
max;      // 200
However if you only want to obtain the min and max value, sorting the whole array is a very inefficient way to do it.

Math.min() and Math.max()

Math.min(numbers)
Math.min() finds the minimum of numbers listed within the brackets.
Math.min(30, 5, 200, 3, 50);                  // 3
Math.min.apply(null, [numbers])
For numbers in array format.
Math.min.apply(null, [30, 5, 200, 3, 50]);    // 3

const numbers = [30, 5, 200, 3, 50];
Math.min.apply(null, numbers);                // 3
Math.max(numbers)
Math.max() finds the maximum of numbers listed within the brackets.
Math.max(30, 5, 200, 3, 50);                  // 200
Math.max.apply(null, [numbers])
For numbers in array format.
Math.max.apply(null, [30, 5, 200, 3, 50]);    // 200

const numbers = [30, 5, 200, 3, 50];
Math.max.apply(null, numbers);                // 200

Made up min max function

Min max function
Fastest solution doesn't have its own built in method. We can write it ourselves as a function.
Max function from w3school
function arrayMaxW3(arr) {
    let max = -Infinity;
    let l = arr.length;
    while (l--) {
        if (arr[l] > max) {
            max = arr[l];
        }
    }
    return max;
}
My arrayMax function
function arrayMax(arr) {
    let max = -Infinity;
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}
My arrayMin function
    function arrayMin(arr) {
        let min = Infinity;
        for (let i = 0; i < arr.length; i++) {
            if (arr[i] < min) {
                min = arr[i];
            }
        }
        return min;
    }
Testing my functions
    const numbers = [30, 5, 200, 3, 50];
    arrayMax(numbers);        // 200
    arrayMin(numbers);        // 3

Object array sort()

Object array sort()
Even when the array consists of object, it can be sorted, based on particular property in the objects.
const kids = [
    {name:"Adam", age:10},
    {name:"Ben", age:4},
    {name:"Chris", age:7}
];

kids.sort(function(a, b){return a.age - b.age});

kids;
// (3) [{name: 'Ben', age: 4}, {name: 'Chris', age: 7}, {name: 'Adam', age: 10}]
