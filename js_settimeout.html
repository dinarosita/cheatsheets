JavaScript: setTimeout()

setTimeout()

setTimeout()
setTimeout(function, milliseconds, ...)
This method calls a function after a number of milliseconds.
Arguments:
  • The function to execute (required)
  • The milliseconds (optional)
  • Other parameters need to be passed in (optional)
Return value:
Id of the timer. Id is for use in clearTimeout(id) to cancel the timer.
clearTimeout()
idName = setTimeout(function, milliseconds);
clearTimeout(idName);
setInterval()
setTimeout() only run the function once. If repeated action needed, use setInterval() instead.
Example: Display greeting after 5 seconds
The greeting
function myGreeting() {
    console.log("Hello Jello!");
}
setTimeout()
const timerHello = setTimeout(myGreeting, 5000);

// After 5 seconds: "Hello Jello!"
Use clearTimeout() to interupt above from running.
const timerHello = setTimeout(myGreeting, 5000);

function stopHello() {
    clearTimeout(timerHello);
  }
  
stopHello();
// After 5 seconds: nothing
