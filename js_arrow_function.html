Arrow Function

functionName = function () {
code block;
return returnValue;
}

functionName = () => {
code block;
return returnValue;
}

Writing function can be shortened by using arrow function where function () {} is written as () => {}.

functionName = () => returnValue;

If the code block only has the return value, it can be even shorter. No curly brackets {} and the word "return" to be used.

hello = function () {
    return "Hi there!";
}                           // same code

hello = () => {
    return "Hi there!";
}                           // same code

hello = () => "Hi there!";  // same code
functionName = (param1, param2) => returnValue;

Just like the a regular function, arguments/parameters are listed inside parentheses ().

multiply = function (a, b) {
    return a*b;
}                           // same code

multiply = (a, b) => {
    return a*b;
}                           // same code

multiply = (a, b) => a*b;   // same code
functionName = param1 => returnValue;

An exception, when there's only 1 parameter, the parentheses () are not needed. Remember when there's no parameter, the parentheses are needed.

greetings = function (userName) {
    return "Hello, " + userName + "!";
}

greetings = userName => "Hello, " + userName + "!";
