Glossary

A

Angle clock: GUI for choosing angle.

API: Application Programming Interface. A set of definitions and protocols for building and integrating application software.

C

Color picker: GUI for changing color

CSS: Cascading Style Sheets

D

DevTools: Developer Tools

DOM: Document Object Model

DOM tree: Document Object Model presented as tree structure.

E

Eyedropper: Color sampler in color picker.

G

GUI: Graphical User Interface. A computer program that enables a person to communicate with a computer through the use of symbols, visual methapors, and pointing devices. GUI is now standard computer interface

H

HTML: HyperText Markup Language

P

Pseudo-class (CSS): Different stages a class can be, for example: :active, :focus, :hover, :visited, :focus-within, or :focus-visible.

U

UI: User Interface

V

Viewport: User's visible area of a web page.
