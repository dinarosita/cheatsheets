HTML Attributes

Attributes

Global attributes

href
URL, absolute or relative
target
Where to show the result of navigation
Values:
_self
Default. Show result in the current browsing context.
_blank
Show result in a new, unnamed browsing context
_parent
Show the result in the parent browsing context. If there's no parents, show in current browsing context just like _self.
_top
Show result in the top most browsing context. If there's no parents, show in current browsing context just like _self.
