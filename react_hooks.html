React Hooks

Basics

Only call hooks from react function

React hooks can only be used inside a function component.

It can't be used in a class component. Class component already has their own thing to do what hooks do in function component.

Function vs class component:

  • Function component:
    function functionName () { ... }
  • Class component:
    class className extends React.component { ... }

Only call hooks at the top level

Read: Hooks rules

Everytime function runs, the hooks must be executed exactly the same order. Only call hooks at top level Meaning they can't be inside loops, conditions, or nested functions.

  • Correct:
    function functionName () { 
    useState()
    useState()
    useState()
}
  • Wrong:
    function functionName () { 
    if (this == something) {
        useState()
    }    
    useState()
    useState()
}

// React hook "useState" is called conditionally. React Hooks must be called in the exact same order in every component render.
