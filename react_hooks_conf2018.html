Problems with current react (2018)

  • Reusing logic - Beacuse you often end up with wrapper hell
  • Giant components - Because you have logic split up across different life(?) cycle
  • Confusing classes for butoh humans and machines.

Rather than 3 different problems, these are actually 3 symptoms of one problem: React doesn't provide a stateful primitive simpler than a class component.
